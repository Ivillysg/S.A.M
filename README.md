## Getting Started
Para iniciar a aplicação, é necessario que tenha o NodeJS instalado em sua maquina.
Se ja tiver o Node, vá até a pasta da aplicação e digite:
* **npm**
```sh
npm install
```
* **yarn**
```
yarn install
```
